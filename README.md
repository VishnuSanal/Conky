# Setting Up

```
curl 'https://raw.githubusercontent.com/VishnuSanal/Conky/main/install.sh' -o install.sh
chmod +x install.sh
./install.sh
```

Add this to autostart:

 ```
conky -c ~/.config/conky/conky_date_time.conf &
conky -c ~/.config/conky/conky_mpd.conf &
 ```
